# Sobre

Este projeto é um sistema web de gerenciamento e acompanhamento de vendas de carros para autônomos, realizado como projeto da faculdade de ADS. 

## Tecnologias e métodos utilizados

- Linguagens: C#
- Estrutura das páginas: Html, Css e Javascript.
- Frameworks: ASP NET. Core 8.0 e Entity Framework.
- Arquitetura: Padrão MVC.
- Banco de dados: MySQL.
- Modelo de Aplicação: Template Engine com Razor.
- Modelos Entidade-Relacionamento: Feito no draw.io
